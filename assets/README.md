# 🖼️ Assets: Visual Proof & Media

This folder serves as the central storage for images, videos, and screenshots used throughout the repository and the final pitch.

## 📁 Contents

### 1. Prototype Photos
- `alpha_prototype_breadboard.jpg`: The functional wiring of the ESP32-S3 and sensor triad.
- `uv_excitation_test.jpg`: Image showing the 365nm UV LED triggering on a turmeric sample.

### 2. Software Screenshots
- `blynk_web_dashboard.png`: View of the real-time quality monitor and "Digital Quality Passport."
- `proteus_simulation_run.png`: Screenshot of the logic analyzer verifying the I2C communication.

### 3. AI Inference Metrics
- `vitis_ai_output.png`: Terminal output from the AMD Kria showing the sub-5 second classification result.

## ⚠️ Usage
If you are contributing to this project, please ensure all new images are optimized for web (under 1MB) and descriptively named.
