# 🌐 Web Dashboard: The Digital Quality Passport

## 📌 Overview
The OmniSense Dashboard is the final interface where "Data becomes Trust." It is designed for two personas: the **Warehouse Manager** (for real-time monitoring) and the **International Buyer** (for quality verification).

## 🛡️ The "Digital Quality Passport" (DQP)
Every successful scan generates a unique **DQP ID**. This ID is a cryptographic hash of the spectral fingerprint, the timestamp, and the AMD Kria's unique hardware ID. 
- **Anti-Tamper:** Once the Kria hub verifies a batch, the data is pushed to the dashboard as a "Locked Entry."
- **Traceability:** Buyers can scan a QR code on the spice container to view the live spectral graph of that specific batch.

## 📊 Visualization Components
- **Spectral Heatmap:** Visualizes the intensity of light across 18 wavelengths (410nm to 940nm).
- **Toxin Indicator:** A dedicated LED widget that turns **RED** immediately if the AI detects Sudan Dye or Aflatoxin.
- **Batch History:** A scrollable table showing the "Purity Score" of every batch processed in the last 30 days.
